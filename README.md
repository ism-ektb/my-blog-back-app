# Бэкэнд блога.

Для сборки и запуска проекта необходима среда разработки и docker.
В корневой папке находится docker-compose для создания базы данных postgresql.
Сборка проекта осуществляется командой "mvnw clean package".

Для эммитации работы базы данных во время тестов используется тестконтейнер, поэтому во время сборки проекта должен быть запущен Docker, либо проверка тестов должна быть отключена.


Проект запускается с помощью программы Tomcat.
Предварительно убедитесь, что в папке webapp Tomcat отсутствует папка root. Если требуется, удалите ее.
После успешной сборки проекта в папке target появится файл ROOT.war. Его надо скопировать в папку webapp Tomcat.
Запустите Tomcat. При первом запуске проект автоматически установится в папку webapp/root.

В корневой директории имеется папака postman с несколькими простыми тестами.
