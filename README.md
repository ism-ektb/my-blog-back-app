# Бэкэнд блога.

Для сборки и запуска проекта необходима среда разработки и docker.
В корневой папке находится docker-compose для создания базы данных postgresql.
Сборка проекта осуществляется командой "gradle clean build".

Для эммитации работы базы данных во время тестов используется тестконтейнер, поэтому во время сборки проекта должен быть запущен Docker, либо проверка тестов должна быть отключена.

После запуска программы по адресу http://localhost:8080/swagger-ui/index.html будет доступна документация проекта
